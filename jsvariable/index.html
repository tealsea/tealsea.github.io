js中定义变量的方法

1、var

基于var创建一个的变量叫N的值，让其指向具体的的值10

var n=10;
console.log(10); // =>10

2、let

基于ES6中的Let创建的一个变量，让其指向100，但后期让其重新指向了一个新值200

let a=100;
    a=200;
console.log(a); // =>200

3、const

基于ES6中的const创建一个变量，让其指向1000，但后期让其指向2000，会报错，因此const创建的值是不能改变，所以const是常量

const b=1000;
      b=2000;
console.log(b); // =>TypeError: Assignment to constant variable.

4、function

创建一个函数，也可以理解为创建了一个fun，让其指向这个函数

function fun(){};
conosle.log(fun); // =>ƒ fun() {}

5、class类

创建一个类也可以理解创建了一个变量Parent,让其指向这个类

class Parent{};
cosnole.log(Parent); // => class Parent { }

6、import/require

基于模块规范来导入具体的某个模块，定义一个axios的变量，用来指向导入的这个模块

// 导入axios模块
import axios from 'axios';
// 导入axios模块
let axios=require('./axions')
